# RemitAI PWA Icons

Please generate icons with the following dimensions from your brand logo using a tool like [RealFaviconGenerator](https://realfavicongenerator.net/).

Required sizes:
- 192x192
- 384x384
- 512x512

Save them as:
- `public/icons/icon-192x192.png`
- `public/icons/icon-384x384.png`
- `public/icons/icon-512x512.png`

Make sure they are "maskable" icons for best results on Android.
